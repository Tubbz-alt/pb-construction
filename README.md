# pb-construction
Pybullet Construction Planning

## Installation

```
$ git clone https://github.com/caelan/pb-construction.git
$ cd pb-construction
$ git submodule update --init --recursive
$ ./pddlstreamFastDownward/build.py
```

## Examples

* `$ python -m extrusion.run`
* `$ python -m picknplace.run`
